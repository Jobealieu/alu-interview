# Pascal's Triangle

## Description
This project implements Pascal's Triangle in Python. Given an integer `n`, the function returns a list of lists representing the first `n` rows of Pascal's Triangle.

## Function
`def pascal_triangle(n):`
- Returns a list of lists of integers
- Returns an empty list if `n <= 0`
- Assumes `n` is always an integer

## Example
```python
pascal_triangle(5)
# [[1], [1,1], [1,2,1], [1,3,3,1], [1,4,6,4,1]]
```
